Bem-vindo ao FuelPHP

O FuelPHP é um framework web PHP 5 conduzido pela comunidade, simples e flexível. Ele nasceu das frustações que as pessoas têm com os atuais frameworks e é desenvolvido com o apoio da comunidade. O FuelPHP é extremamente portátil, funciona em praticamente qualquer servidor e se orgulha de sua sintaxe limpa.

Instalação Rápida

Se você valoriza o seu tempo e quer ter as coisas funcionando em segundos em vez de minutos, você pode usar o nosso instalador. Usando a biblioteca curl, você pode instalar uma versão básica do oil. A partir daí, você pode criar novas aplicações usando o Fuel.

# instalação rápida do oil a partir da web
$ curl get.fuelphp.com/oil | sh
# agora que o oil está instalado, crie um projeto de blog em um diretório chamado Sites
$ cd Sites/
$ oil create blog

Para mais informações, veja a página de instalação.

Instalação Manual

Se você não puder usar o método de instalação rápida, você também pode instalar manualmente o FuelPHP. Você irá encontrar instruções detalhadas na página de instalação.
