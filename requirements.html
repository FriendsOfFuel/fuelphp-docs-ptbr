Requisitos

Servidor Web
  • Qualquer servidor web. O Fuel foi testado no Apache, no IIS e no Nginx.
  • Funciona em todos os principais sistemas operacionais, incluindo os *nix e o Windows.
Versão do PHP
  • É necessária a versão 5.3 ou maior.
Extensões PHP

O Fuel utiliza funcionalidades que o PHP define como "extensões". Para evitar possíveis problemas de compatibilidade, o Fuel usa apenas extensões que fazem parte do código do PHP, como é documentado no site do PHP. Se forem usadas extensões que sejam parte do repositório PECL (que precisam ser instaladas manualmente por um administrador do sistema), o Fuel irá utilizá-las se estivem disponível, caso contrário usará uma solução alternativa.

Essa é a lista das extensões usadas, e das classes que as utilizam:

Extensão Usada em Comentários
fileinfo Upload Documentação.
Observe que, em ambientes Windows, a instalação manual de uma DLL pode ser necessária.
mbstring n/a Documentação.
Se não estiver disponível, o framework não suportará multibyte. Você pode verificar a constante MBSTRING para descobrir se sua instalação tem suporte habilitado a multibyte.
mcrypt Crypt Documentação.
Se não estiver disponível, a PHPSecLib é usada para emular sua funcionalidade.

Recomendações

  • Servidor Web - Apache rodando em um *nix.
  • Versão do PHP - 5.3.2
  • mod_rewrite - Para remover o index.php das URLs.
  • PHPUnit - 3.5.12
